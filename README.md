# 🛴kick-cops


## 프로젝트 구성
* 전동 킥보드 불법 주정차 민원을 줄이고, 인도에 전동 킥보드 무단 방치를 줄이기 위해서
* 구체적으로 킥보드가 인도 위에 방치돼서 발생하는 통행 혼잡 및 안전 문제 해결
* 프로젝트 수행을 위해 알아야 할 것
* 1. 강서구에서 견인된 전동 킥보드 현황 파악
  2. 전동 킥보드 감지 모델 구현 방법
  3. 바른 주차 시스템 구현 방법
  4. 기업에 신고하는 방법
